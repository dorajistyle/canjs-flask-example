안녕하세요?

제안소 소스의 간단한 안내입니다.

1. 라이브러리 설치
우선 아래 명령어를 이용하여, 필요 라이브러리를 설치하여 줍니다.
실행을 위해 pip 라이브러리가 설치되어 있어야 합니다.
--------------------------------------------------------
pip install -r requirements.txt
--------------------------------------------------------

혹시 설치가 제대로 되지 않는다면 아래 모듈을 수동으로 설치하여 주십시오.
--------------------------------------------------------
Flask==0.9
Flask-SQLAlchemy==0.16
Jinja2==2.6
pyjade==2.0.2
flask-babel==0.8
flask-testing==0.4
blinker==1.2
--------------------------------------------------------


2. DB 생성

프로젝트 폴더에서 다음을 실행합니다.

--------------------------------------------------------
python init.py
--------------------------------------------------------


3. 어플리케이션 실행

프로젝트 폴더에서 다음을 실행합니다.

--------------------------------------------------------
python 프로젝트 폴더에서 다음을 실행합니다.

--------------------------------------------------------
python canjs_flask_proposal.py
--------------------------------------------------------

4. 어플리케이션 설명
1) 화면 상단의 주황색 버튼을 누르면, 제안하기 페이지로 갑니다.
2) 분류를 선택하고, 이메일, 내용, 개인정보이용 동의 후 보내기를 누르면 DB에 제안이 저장됩니다.
3) 목록 화면에서 추천 버튼을 누르면 추천이 됩니다.
4) 목록이 3개 이상일 때 리스트 화면에서 '더 보기' 버튼을 누르면, 다음 제안 목록을 더해 보여줍니다.
5) 언어의 변경은 파라메터로 합니다. (?lang=en|ko)

이상입니다.
